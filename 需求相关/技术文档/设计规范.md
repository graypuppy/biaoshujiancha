# 标桥产品 - 视觉设计规范 (简约版)

> **适用场景**: 用于AI生成产品页面时,作为通用的视觉设计要求部分

---

## 🎨 核心设计规范

### 一、设计风格定位
- **产品**: 标桥标书检查系统,面向大中型建筑企业
- **风格**: 现代高效、科技感、专业可信
- **视觉特点**: 微立体渐变、简洁克制、信息密度适中

---

### 二、色彩系统

#### 品牌色
```css
主色: #125FFA (标桥蓝)
主色渐变: linear-gradient(135deg, #125FFA 0%, #0A47C7 100%)
辅助色: #ECF5FF (浅蓝)
```

#### 中性色
```css
主文字: #1A1C24
正文: #3D3D3D
次要文字: #6B6B6B
禁用/占位: #BFBFBF
页面背景: #F5F7FA
卡片背景: #FFFFFF
分割线: #E8EAED
```

#### 功能色
```css
成功/通过: #3CB230
警告/提醒: #E98200 / #FAAD14
错误/紧急: #E4231D
信息: #125FFA
特殊: #722ED1 (紫)
```

#### 项目状态色 (常用)
```css
已创建: #8C8C8C (灰)
制作中: #125FFA (蓝)
检查中: #FAAD14 (黄)
已提交: #3CB230 (绿)
已开标: #722ED1 (紫)
已中标: #2D8A23 (深绿)
```

---

### 三、字体规范

#### 字体族
```css
font-family: "PingFang SC", -apple-system, "Microsoft YaHei", sans-serif;
```

#### 字号层级
```css
大标题: 28px / 行高40px / 字重600
标题: 20px / 行高32px / 字重600
小标题: 16px / 行高24px / 字重600
正文: 14px / 行高22px / 字重400
辅助文字: 12px / 行高18px / 字重400
```

#### 字重
- 400 Regular: 正文内容
- 500 Medium: 强调信息
- 600 Semi-Bold: 标题、按钮

---

### 四、基础样式

#### 圆角
```css
小圆角: 2px (输入框内图标)
标准圆角: 4px (按钮、输入框、卡片)
中圆角: 8px (大卡片、对话框)
大圆角: 12px (徽章)
圆形: 50% (头像)
```

#### 阴影
```css
卡片: 0 2px 8px rgba(0, 0, 0, 0.08)
悬浮: 0 4px 12px rgba(18, 95, 250, 0.15)
弹窗: 0 8px 24px rgba(0, 0, 0, 0.15)
按钮: 0 2px 4px rgba(18, 95, 250, 0.2)
```

#### 间距 (8px栅格)
```css
极小: 4px
小: 8px
标准: 12px / 16px
大: 24px
特大: 32px / 40px
```

---

### 五、核心组件样式

#### 主按钮
```css
背景: linear-gradient(135deg, #125FFA, #0A47C7)
文字: #FFFFFF, 14px, 600
圆角: 4px
内边距: 8px 24px
高度: 36px
投影: 0 2px 4px rgba(18, 95, 250, 0.2)

Hover: 投影加深 + 向上-1px
```

#### 次要按钮
```css
背景: #FFFFFF
文字: #3D3D3D, 14px, 500
边框: 1px solid #D9DCDF
圆角: 4px

Hover: 背景#ECF5FF + 边框#125FFA + 文字#125FFA
```

#### 输入框
```css
背景: #FFFFFF
边框: 1px solid #D9DCDF
圆角: 4px
高度: 36px
内边距: 8px 12px
占位符: #BFBFBF

Focus: 边框#125FFA + 外发光 0 0 0 2px rgba(18, 95, 250, 0.2)
```

#### 卡片
```css
背景: #FFFFFF
圆角: 8px
投影: 0 2px 8px rgba(0, 0, 0, 0.08)
内边距: 24px

Hover: 投影 0 4px 12px rgba(18, 95, 250, 0.15) + 上浮-2px + 边框#125FFA
```

#### 状态徽章
```css
高度: 24px
内边距: 4px 12px
圆角: 12px (胶囊型)
字号: 12px, 500
背景与文字根据状态使用对应颜色
```

#### 表格
```css
表头背景: #FAFBFC
行高: 48px
内边距: 12px 16px
斑马纹: 偶数行#FAFBFC
边框: 1px solid #E8EAED

Hover行: 背景#ECF5FF
```

### 六、交互动效

#### 过渡时长
```css
快速: 0.15s (小元素)
标准: 0.3s (常规交互)
缓慢: 0.5s (大面积变化)
缓动: ease-in-out
```

#### 常用动画
```css
悬浮上浮: transform: translateY(-2px)
缩放: transform: scale(1.05)
淡入淡出: opacity 0.3s
```

#### 栅格系统

## ✅ 使用检查清单

生成的设计需满足:
- [ ] 使用了品牌色#125FFA作为主色
- [ ] 文字颜色层次分明(主/正文/次要)
- [ ] 卡片有适当的投影和圆角
- [ ] 按钮有渐变背景

- [ ] 间距符合8px栅格
- [ ] 整体风格现代、专业、有科技感

---

## 💡 设计原则

1. **克制**: 不过度使用颜色和装饰
2. **一致**: 同类元素保持统一样式
3. **对比**: 重要信息通过颜色和大小强化
4. **呼吸**: 留白充足,信息不拥挤
5. **反馈**: 所有交互有视觉响应

## 🔧 常见场景快速应用

### 场景1: 列表/表格页面
```
使用: 
- 白色卡片包裹表格
- 表头浅灰背景#FAFBFC
- 斑马纹增强可读性
- Hover行浅蓝背景#ECF5FF
- 状态用彩色徽章标识
```

### 场景2: 表单页面
```
使用:
- 输入框统一高度36px
- 必填项标题加红色*
- 提交按钮用主色渐变
- 表单分组用分割线或留白
```

### 场景3: 数据看板
```
使用:
- 关键数据大号字体 + 渐变色
- 卡片网格布局,间距16px
- 趋势用绿色↑ / 红色↓
- 图表用主色系渐变
```

### 场景4: 空状态
```
使用:
- 中央大号图标(80px灰色)
- 提示文字14px灰色
- 行动按钮主色渐变
- 整体垂直居中
```
